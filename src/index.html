                          Console

                          Use the console for input. Here are a list of available commands:

                          To add a note, simply type it in and press Enter:

                          Glaciers are cool

                          Fire burns

                          To add a task, append it with a + sign. Clicking on a task marks it as completed.

                          + Buy food

                          + Pick up the mail

                          To add an event, specify a time or date after an @ symbol. Remember to write dates in full if you choose to use them. Times are to be set in 24-hour format.

                          Presentation @ today 9:00

                          Museum @ next Tuesday 15:00

                          Road trip @ December 23 2017

                          Use tags to categorise and group related notes, tasks, and events.

                          The #coffeehouse is closed on Saturdays

                          + Feed the #cat

                          Lunch #meeting @ tomorrow 12:00

                          If you have an existing data file (JSON), you can import it:

                          import

                          To export your data:

                          export
